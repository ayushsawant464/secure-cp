## During the Exam

### What Happens When Exam Starts

1. **Administrator starts the system**
   - Your screen will go fullscreen
   - You'll see only the exam website
   - Your keyboard and mouse will work normally within the browser

2. **What you CAN do**:
   - ‚úÖ Navigate within the exam platform
   - ‚úÖ Write and submit code
   - ‚úÖ Use all features of the exam website
   - ‚úÖ Scroll, click, type normally

3. **What you CANNOT do**:
   - ‚ùå Exit the browser
   - ‚ùå Open other applications
   - ‚ùå Access other websites
   - ‚ùå Use keyboard shortcuts (Alt+Tab, Alt+F4, etc.)
   - ‚ùå Switch to terminal or file manager

### Normal Exam Behavior

**The browser will**:
- Stay in fullscreen mode
- Only show the exam website
- Block access to other sites
- Disable developer tools (F12)
- Disable right-click menu

**Your keyboard will**:
- Work normally for typing
- NOT respond to Alt+F4, Alt+Tab, etc.
- NOT allow virtual console switching (Ctrl+Alt+F1-F6)

**Your network will**:
- Only access the exam website
- Block all other websites
- Route through a secure VPN

## What NOT to Do

### ‚ùå DO NOT Attempt These Actions

| Action | Why Not | Consequence |
|--------|---------|-------------|
| **Press Alt+F4** | Won't work, will be logged | Flagged as escape attempt |
| **Press Alt+Tab** | Won't work, will be logged | Flagged as escape attempt |
| **Press Ctrl+Alt+F1** | Won't work, will be logged | Flagged as escape attempt |
| **Unplug network cable** | Exam will pause | Time lost, flagged |
| **Restart computer** | Exam will end | Must restart exam, time lost |
| **Try to open terminal** | Won't work, will be logged | Flagged as cheating attempt |

### Why These Are Blocked

The system is designed to ensure:
- **Fairness**: Everyone has the same environment
- **Security**: No unauthorized resources
- **Integrity**: Your work is your own

## Understanding the Security

### Why Is the System Locked Down?

The Secure Exam System ensures:
1. **Fair testing environment** for all students
2. **Academic integrity** - your work is your own
3. **Prevents cheating** - no unauthorized resources
4. **Protects exam content** - questions stay confidential

### What Is Being Monitored?

- ‚úÖ Which applications you try to run
- ‚úÖ Which websites you try to access
- ‚úÖ Any attempts to exit the browser
- ‚ùå NOT your keystrokes or screen content
- ‚ùå NOT your webcam (unless separate proctoring)

### Your Privacy

- Logs are kept for academic integrity only
- Only authorized staff can access logs
- Logs are deleted after retention period
- No personal data is collected beyond what's necessary
